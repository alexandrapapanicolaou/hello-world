Description: This README.txt contains the instructions on how to run the executable create_project_folder_structure.sh

Instructions:

	1. To run this script open either your terminal and navigate to the folder where you want to create this new folder structure.

	2. From the command line type in the following: sh ~/scripts/create_project_folder_structure.sh <name of project>. For example, if you want to create a new folder Project_test you would type
	sh ~/scripts/create_project_folder_structure.sh Project_test

	3. After the program runs, you should have your folder in the desired location.

	4. Alternatively, if you don't want to run steps 1 to 3, just copy and paste the folder folder_template into your desired location and change its name
